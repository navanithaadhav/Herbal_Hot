Herbal_Hot is an e-commerce web application for browsing and buying spices/herbal products. It consists of a React + TypeScript + Vite frontend (customer shop + admin dashboard) and a Node.js + Express + TypeScript backend exposing REST APIs for users, products, orders, Razorpay payment configuration, and image uploads. Core integrations include Google OAuth (frontend + backend) and Razorpay checkout/payment verification, plus Cloudinary-backed image uploads and JWT-based auth/role guards.
